Setup for the Python3.8 in Ubuntu 16.04 environnment

Note: If you have better solutions, or there is any operations not correct, please let the TAs know.

Installing Python3.8

Downloading

  • As deadsnakes is not supported for Ubuntu 16.04, the present useful way I found is using the source code on Python.org and find the certain Python3.8 version.

  • Choose the tar.gz source code and download it.

Dependencies

  • Add these dependencies in case of error in the following steps.

Install

  • Open the folder where you download it, extract it and enter the folder.
  • Configure and Make

Add sudo due to the error of permission denied

Add Links

  • Use the command to add link for the python3.8
  • Although it is not correct, it can fix the problems in the last step of packages update

    The python.bak is the python2 link.

More later…

  • Now you can check the python version. You can see Python3.8+.

  • Follow the other steps shown in the tutorial to finish the setup.

Solution to the error checking Ryu

  • Some of you may find it error when trying to check the Ryu in the last step. It is because of the version of package eventlet.

  • To fix it, install another version of it.

  • It should be fine then.
